Course Overview


This course will discuss the basics and advance concepts about phishing, open redirect, and reverse tabnabbing vulnerabilities. This course will also cover discussions on how attackers/hackers weaponize these vulnerabilities to phish victims. But most importantly, this course will provide a guide for developers on how to secure their websites from the said vulnerabilities.

Internet Protocol (IP) addressing will also be discussed here and how browsers handle different IP addresses notations and bypasses, and unicode characters in uniform resource locators (URLs).

By the end of this course, you will be able to make your own URL validation library that uses PHP and JavaScript languages to help your websites safe from phishing, open redirect, and reverse tabnabbing attacks.
