Making a Reverse Tabnabbing-Prevention JavaScript Library


Let's make the reverse tabnabbing prevention library in JavaScript.

Protection against reverse tabnabbing only needs 2 steps, find all anchor tags and add the nofollow/noopener/noreferrer.

1. Find all anchor tags that has a target property equal to _blank. This can be done using document.querySelectorAll.
let anchorTags = document.querySelectorAll('a[target=_blank]');

2. Iterate to each anchor tag add add the "rel" property with a value of “noopener noreferrer nofollow”.
for(let i = 0; i < anchorTags.length; i++){
    anchorTags[i].setProperty('rel', 'noopener noreferrer nofollow')
}

3. We just need to place it on a function so we can easily call it anywhere we want.
const protectAnchorTags = () => {
    let anchorTags = document.querySelectorAll('a[target=_blank]');
    for(let i = 0; i < anchorTags.length; i++){
        anchorTags[i].setProperty('rel', 'noopener noreferrer nofollow')
    }
}
